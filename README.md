# Local RAG PDF Assistant

A Retrieval-Augmented Generation (RAG) system for answering questions from PDF documents using local LLMs.

## Tech Stack

- Python
- Ollama
- Qwen 2.5
- ChromaDB
- Sentence Transformers
- PyMuPDF
- Streamlit

## Architecture

PDF → Extraction → Chunking → Embeddings → ChromaDB → Retrieval → Local LLM → Answer
